# 🧪 BNS Pharma Platform

A full-stack pharmaceutical platform built with Next.js, Prisma, and modern UI/UX practices.  
Designed for institutional-grade healthcare product discovery, filtering, and management.

---

## 🚀 Features

- 🔍 Advanced product search (nuqs-based filtering)
- 📦 Paginated product catalog
- 🧑‍⚕️ Role-based authentication (Admin/User)
- ⭐ Ratings & reviews system
- 📩 Contact form with email integration (Resend)
- ⚡ Optimized caching with Next.js server actions
- 🎨 Modern UI (ShadCN + Tailwind + Glassmorphism)

---

## 🛠 Tech Stack

- **Frontend:** Next.js 14 (App Router), React, TailwindCSS
- **Backend:** Server Actions, Prisma ORM
- **Database:** PostgreSQL
- **Auth:** NextAuth.js
- **Email:** Resend
- **State Management:** React Hooks
- **Validation:** Zod
- **UI:** ShadCN UI + Lucide Icons
